4. Using a Relay

Step-by-Step Guide: Control High Voltage with a Relay

A relay lets your ESP8266 turn on or off a device that runs on higher voltage, like a lamp or motor.


What is a Relay?

A relay is an electrically controlled switch. You can use a small signal from the ESP8266 to control a larger electrical load.

  • ESP8266 controls the relay using a digital pin
  • The relay switches ON/OFF a high-power device
  • Great for smart home projects!

Step 1: Wiring the Relay

Connect the relay module to the ESP8266 D1 Mini like this:

  • VCC → 5V
  • GND → GND
  • IN → D1 (GPIO5)

NOTE: Many relay modules are active LOW, meaning LOW turns them ON.


Step 2: Test Code

This code turns the relay ON for 1 second and OFF for 1 second in a loop.

C++ Relay Test 
#define RELAY_PIN D1 // GPIO5

void setup() {
  pinMode(RELAY_PIN, OUTPUT);
}

void loop() {
  digitalWrite(RELAY_PIN, LOW);  // Relay ON
  delay(1000);
  digitalWrite(RELAY_PIN, HIGH); // Relay OFF
  delay(1000);
}

What Does It Do?

  • pinMode() sets D1 as an OUTPUT pin
  • LOW turns the relay ON (activates the switch)
  • HIGH turns the relay OFF
  • delay(1000) waits 1 second between each switch

Now you can hear a "click" every second. That’s the relay switching!


⚠️ Safety Warning

If you're using the relay to control 230V (AC power), follow these rules:

  • Never touch the relay when it’s powered
  • Use heat-shrink tubing or enclosures
  • Ask a teacher or adult if you’re unsure

Always double-check your wiring and be safe!
