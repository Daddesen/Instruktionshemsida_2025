2. Installing Arduino IDE

To upload code to your ESP8266 D1 Mini, you need a program called the Arduino IDE. It allows you to write code and send it to the microcontroller.


1. Download the Arduino IDE

Go to the official Arduino website and download the software for your operating system:
https://www.arduino.cc/en/software

  • Choose the right version: Windows, Mac, or Linux.
  • Install it like any normal program.

2. Add Support for ESP8266 (D1 Mini)

The Arduino IDE doesn’t support the ESP8266 by default, so we need to install it manually. Here's how:

  1. Open the Arduino IDE.
  2. Go to File → Preferences.
  3. Find the field called "Additional Board URLs".
  4. Paste in this URL:
    https://arduino.esp8266.com/stable/package_esp8266com_index.json
  5. Click OK.

3. Install the ESP8266 Board

  1. Go to Tools → Board → Boards Manager.
  2. Search for esp8266.
  3. Click Install on the package called “esp8266 by ESP8266 Community”.

4. Select the Right Board

After installation, go to:

  • Tools → Board and select “LOLIN(WEMOS) D1 mini (clone)”.
  • This is the correct board setting for your ESP8266 D1 mini.

5. Connect Your Board

  • Use a USB cable to connect your D1 mini to your computer.
  • Go to Tools → Port and select the port that appears (usually COM3 or higher).

You’re now ready to upload your first program!


What does this code mean?

C++ Random example? NO! 

void setup() {
  // put your setup code here, to run once:
}

void loop() {
  // put your main code here, to run repeatedly:
}

Explanation

This is a basic structure of an Arduino program, also called a "sketch". It consists of two main functions:

  • setup(): This function runs once when the microcontroller starts. It is used to set up things like pin modes, serial communication, or initializing libraries. In this example, it is empty.
  • loop(): This function runs continuously in a loop after setup() is done. It contains the main logic of your program. This is where you would put the code that needs to run repeatedly. In this example, it is also empty.

This structure is required for all Arduino programs, even if they don’t do anything yet.
